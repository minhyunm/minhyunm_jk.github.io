TDD Study : week 0

유용한 TDD 개념

  • 사용자 스토리(User story) : 사용자 관점에서 어떻게 애플리케이션이 동작해야 하는지 기술한 것. 기능 테스트 구조화를 위해 사용
  • 예측된 실패(Expected failure) : 의도적으로 구현한 테스트 실패
  • 기능 테스트 : 사용자 관점에서 애플리케이션 외부를 테스트
  • 단위 테스트 : 프로그래머 관점에서 애플리케이션 내부를 테스트
